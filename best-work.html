Best Work

Interactive Cooking App

This is an example of my best work because I coded a part of my project with skills that we did not learn in class. An example of this is my drop down menu bar. During this project, we had not yet learned how to have certain things pop up when you choose different options in the menu bar. Because of this I truly think this is my best work because it took me days and hours to figure out how to do it. Although this seems minimal, I think it stands out because it shows that this class has set me up to be someone who figures out how to code the things she wants without anyone having to teach her beforehand. Another thing that I am proud of is the design of this project. I think that the design, specifically, ties everything together because it makes it visually appealing. I found this project hard to do because of how long it took me to figure out how to make certain things pop up when you choose a certain drop down menu option. However, it was also easy because I feel like we learned a lot of the things that I used in my project. I used my knowledge and the skills I learned from all of the code.org levels we did. I truly think that without the code.org I would not be able to make this project. Code.org provided me with a pathway or some sort of “guide line” that allowed me to create this project.
