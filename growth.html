Growth

Algorithim App

This work represents an area of growth for me because the way I designed my apps has changed significantly. Before, I was not able to fully decorate any type of project that I had to code for simply because I did not know what type of code to type in order to get the design I wanted. Additionally, I believe that this work represents an area of growth for me because I learned how to create an app that used an algorithm. Most of the coding projects I worked on before this project were more simple meaning that the app ran mostly on clicking buttons. However, this project not only ran on clicking random buttons but it also gave you an outcome that was designed specifically for you based on the buttons you picked. The hardest area for me to master was the algorithm because I had to create some sort of “rule” where you got a certain outcome because of the buttons. This was hard for me because I had to make each button a certain amount of points and make sure that they are all equal. An obstacle I faced was that there was one specific outcome that was more favorable than the other. In order to get past this obstacle, I needed to figure out a way to make sure that that specific outcome didn’t overpower the rest of the outcomes. This project directly influenced my success because I think it set me up for a higher level of CS knowledge. I say this because this project was one of my more advanced projects and having this challenge has pushed me to learn more about CS, overall allowing me to grow in this class. Something I would change would be to add more outcomes. In this project I did the minimum of what was asked of me and I feel like with the person I am now with so much more CS knowledge, I think I am capable of adding more outcomes to make it more complex.
